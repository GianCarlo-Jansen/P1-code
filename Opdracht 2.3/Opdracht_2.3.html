Opdracht 2.3

1. Waarom noem je de eerste HTML-pagina; index.html?

"Index.html" is de standaardnaam voor de startpagina van een website.

2. Met welke HTML versie werken we nu?

Momenteel werken we met HTML5.

3. Met welke CSS versie werken we nu?

Op dit moment gebruiken we CSS3.

4. Hoe kan het dan een website er net iets anders uitzien in een andere browser?

Websites zien er soms een beetje anders uit in verschillende browsers omdat elke browser zijn eigen manier heeft om de dingen op het scherm te laten zien. Dit kan kleine verschillen in hoe de website eruitziet veroorzaken.

5. Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Op dit moment is Google Chrome de meest populaire webbrowser.
